# Poring Bot Legal Documentation

Welcome to the official legal documentation repository for **Poring**, a powerful Discord logging and moderation utility bot designed to help communities maintain transparency, security, and accountability.

This repository contains all legal, privacy, and compliance policies related to Poring and its services.

## About Poring

Poring is a logging-focused Discord bot designed to provide detailed server event tracking. The bot helps server administrators monitor important activities such as moderation actions, role changes, message edits, channel updates, and other critical events within their communities.

The goal of Poring is to provide a reliable, secure, and transparent logging system that helps server moderators maintain control and visibility over server activity.

## Contents of this Repository

This repository includes the following documentation:

- Privacy Policy
- Terms of Service

- Cookie Policy

These documents explain how Poring handles user data, what responsibilities server administrators have when using the bot, and the standards we follow to maintain user privacy and security.
