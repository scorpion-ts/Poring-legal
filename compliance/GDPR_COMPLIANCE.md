# GDPR Compliance

Last Updated: 2026

Hawk™ is committed to respecting user privacy and protecting personal data in accordance with general data protection principles.

Although Hawk™ primarily operates within the Discord platform, the project follows best practices inspired by the General Data Protection Regulation (GDPR) to ensure responsible handling of user data.

## Data Processing Principles

Hawk™ follows the following principles when processing data:

- Data is collected only when necessary for the operation of the logging system.
- Data collection is limited to technical identifiers such as server IDs, user IDs, and channel IDs.
- Data is processed solely for the purpose of providing logging and moderation features.

## Lawful Use of Data

All information collected by Hawk™ is used strictly for operational functionality. Data is never sold, rented, or shared for marketing purposes.

## Data Minimization

Hawk™ only stores the minimum amount of data required for its logging functionality.

No sensitive personal information such as passwords, private conversations outside servers, or financial data is collected.
