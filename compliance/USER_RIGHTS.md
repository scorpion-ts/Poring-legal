# User Rights

Last Updated: 2026

Users and server administrators interacting with Hawk™ have certain rights regarding their data.

## Right to Transparency

Users have the right to understand what type of data Hawk™ collects and how it is used.

Documentation in this repository provides detailed explanations of the bot's data handling practices.

## Right to Removal

Server administrators may remove Hawk™ from their servers at any time.

Upon removal, stored configuration and related data may be deleted from the system within a reasonable timeframe.

## Right to Inquiry

Users may contact the developer to request clarification regarding how the bot processes data.
