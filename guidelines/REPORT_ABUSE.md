# Report Abuse

Last Updated: 2026

Poring encourages responsible usage and community safety.

If you believe Poring is being abused or used in a harmful manner, you may report the issue to the developer.

## Examples of Abuse

Abuse may include:

- Attempting to exploit vulnerabilities
- Using the bot for harassment or malicious activity
- Automated spam or service disruption
- Unauthorized attempts to access system infrastructure

## Reporting Method

To report abuse, please provide as much information as possible, including:

- Server ID
- Description of the issue
- Screenshots or evidence if available
