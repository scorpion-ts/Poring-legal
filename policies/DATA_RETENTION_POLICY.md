# Hawk™ Data Retention Policy

Last Updated: 2026

This document explains how Hawk™ handles and stores data collected through its logging system.

## Data Storage

Hawk™ stores server activity logs in order to provide moderation insights and event tracking.

Stored data may include:

- Server event logs
- Channel changes
- Role updates
- Moderation actions
- Message edit or deletion logs

## Retention Period

Data is stored only for as long as necessary to maintain the functionality of the logging system.

Server administrators may clear stored data by removing the bot or resetting its configuration.

## Data Removal

When Hawk™ is removed from a server, associated stored data may be deleted from the system within a reasonable time frame.

## Policy Updates

Retention policies may change in the future to improve system performance or comply with legal requirements.
