# Poring Security Policy

Security is a top priority for Poring.

## Reporting Vulnerabilities

If you discover a security vulnerability in Poring, please report it responsibly to the developer.

Please do not publicly disclose vulnerabilities before they are addressed.

## How to Report

You can report vulnerabilities through the following methods:

Discord: scorpion.ts  
GitHub: scorpion-ts

## Responsible Disclosure

Security researchers are encouraged to report issues responsibly and provide sufficient information to reproduce the vulnerability.

The developer will investigate and resolve security issues as quickly as possible.
