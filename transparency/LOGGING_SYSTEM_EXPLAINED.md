# Logging System Explained

Last Updated: 2026

Poring provides a comprehensive logging system designed to track important activities within Discord servers.

## What the Logging System Does

The logging system records server events such as:

- Message edits and deletions
- Channel creation or deletion
- Role updates
- Member joins or leaves
- Moderation actions

These logs help administrators monitor server activity and maintain community safety.

## Purpose

The goal of the logging system is to increase transparency within communities and assist moderators in identifying suspicious or harmful actions.

## Configuration

Server administrators can configure which events are logged through Poring settings and commands.
