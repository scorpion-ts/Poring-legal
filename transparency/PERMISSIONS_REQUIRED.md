# Permissions Required

Last Updated: 2026

Poring requires certain Discord permissions in order to function correctly.

These permissions allow the bot to observe server events and send log messages.

## Common Permissions

Depending on configuration, Poring may require:

- View Channels
- Send Messages
- Embed Links
- Read Message History
- Manage Webhooks

## Why These Permissions Are Needed

These permissions allow Poring to monitor server events and deliver logging information to designated log channels.

The bot does not use permissions for purposes unrelated to its logging functionality.

## Administrator Control

Server administrators maintain full control over the permissions granted to Poring and may modify them at any time.
